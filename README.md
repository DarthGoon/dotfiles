dotfiles
========
| Vim Command | Action |
| --- | --- |
| ,d | Toggle file tree |
| ,v | Toggle paste mode |
| ,h  | Clear highlighted matches |
| ,p | Fuzzy file find |
| ,a | Ack |
| ,s | Syntax check |
| ,i | Toggle tabs / spaces |
| ,p | Paste (preserve indentation) |
| ,y | Yank entire file |
| ,c[space] | Toggle comment block |
| ,t | New tab |
| ,[tab] Next tab |
| ,q Close tab |
| ,[1-9] Navigate to tab __num__ |
[ ,0 Navigate to last tab |
| ,gd | `git diff` |
| ,gs | `git status` |
| ,gl | `git log` |
