dotfiles
========
| Vim Command | Action |
| --- | --- |
| ,v | Toggle paste mode |
| ,h  | Clear highlighted matches |
| ,p | Fuzzy file find |
| ,a | Ack |
| ,s | Syntax check |
| ,t | Toggle tabs / spaces |
| ,p | Paste (preserve indentation) |
| ,y | Yank entire file |
| ,c[space] | Toggle comment block |
| ,gd | `git diff` |
| ,gs | `git status` |
| ,gl | `git log` |
